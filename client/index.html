Fizz Buzz game

According to wikipedia this game is used to understand the concept of divisibility.
So, someone says a number and you respond according to following rules:
  • If it is divisible by 3 you say "fizz"
  • if it is divisible by 4 you say "buzz"
  • if it is divisible by both you answer "fizzbuzz"
  • otherwise you repeat the number
